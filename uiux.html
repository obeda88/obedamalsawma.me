Design Decisions
01
Pixel-native typography
All UI text uses a monospaced pixel font, matching the game's art style. Labels like SKILL / DESC / HEALTH keep a consistent grid structure, making info easy to scan mid-game.
02
Character stat card system
Each card encodes name, health, skill, description, and cooldown in a compact layout. Colour hue shifts per character create instant visual identity without breaking the grid.
03
Icon clarity at small sizes
The 7 character select icons are readable at ~64×64px — key silhouettes and colour were prioritised over fine detail so players can identify characters at a glance.
